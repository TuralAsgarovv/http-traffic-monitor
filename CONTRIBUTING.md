# Töhfə Vermək Qaydaları (Contributing)

Bu layihəyə maraq göstərdiyiniz üçün təşəkkür edirik! Biz açıq mənbəli icmanın gücünə inanırıq.

## Necə kömək edə bilərsiniz?

1. **Xəta Bildirişi (Bug Reports):** Əgər kodda bir səhv tapsanız, zəhmət olmasa "Issue" bölməsində ətraflı şəkildə bildirin.
2. **Yeni Xüsusiyyətlər:** Yeni ideyalarınız varsa, müzakirəyə açın.
3. **Kod Dəyişiklikləri (Pull Requests):**
    - Layihəni `fork` edin.
    - Yeni bir `branch` yaradın (`git checkout -b feature/yeni-funksiya`).
    - Dəyişiklikləri edin və testləri yoxlayın.
    - Kodunuzu `push` edin və Pull Request göndərin.

## Kod Standartları

- Dəyişən adları aydın və ingilis dilində olmalıdır.
- Şərhlər (comments) Azərbaycan dilində yazılmalıdır.
- `PEP 8` standartlarına riayət edin.
- Hər yeni funksiya üçün `tests/` qovluğunda test yazın.

Uğurlar!
